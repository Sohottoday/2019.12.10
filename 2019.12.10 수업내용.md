### printf

* System.out.println(), System.out.print()

* - 출력 데이터를 한개만
  - 출력 데이터가 여러개일 때는 + 연산자로 결합하여 한 개로 전달해야 한다.
  - 출력하려는 데이터의 타입에 따라 출력 형식이 정해진다.

  int -> 10진수 숫자 / char -> 문자 / double -> 소수점 이하의 자리수를 설정 불가

- Java5(JDK 1.5) 이후부터

  System.out.printf();

  System.out.printf("출력하려는 포맷 문자열")

  System.out.printf("가나다 \n") or System.out.printf("가나다%n")

  줄바꿈이 안되므로 위의 방식으로 \w 혹은 %n을 붙여줘야 줄바꿈이 된다.

  System.out.rpintf("가%d나%.3f다", 100, 99.56789)

  - 수식이나 변수가 아닌 문자열만 처음에 입력이 가능하다.
  - 수식같은것을 넣고 싶으면 넣고싶은 위치에 % 후 수식
  - 뒤에는 출력할 실제 데이터 값
  - 출력할 값과 포맷문자의 타입이 맞지 않으면 에러가 뜬다.

  %d : 덧셈   //   %.2f : 소수점 2번째자리까지

  %#x : 8진수로 표현   //   #o\ : 16진수   //   %f : 부동 소수점 자리로 표현(6자리)

  %e : 지수 형태로 표현   //   %.4f : 소수점 몇째자리까지 표시

  %s : 길이만큼 알아서 출력(%10s는 앞에 공간 / %-10s는 뒤에 공간)

  %10s -> 10개의 문자가 들어갈 만큼 공간 획득하라(java의 경우 4글자이므로 6자리는 공백)

  %,d = 천 단위마다 콤마 넣기

```java
System.out.print("테스트입니당\n");	//테스트입니당
System.out.print("%d%x%o%c\n", 100, 100, 100, 100);	//100 64 144 d
System.out.print("%#x %#X %#o\n", 100, 100, 100);	//0x64 0X64 0144
System.out.print("%c %c %c %c\n", '가', ''A', '!', '3');	//가 A ! 3
System.out.print("%b\n", true);	//true
System.out.print("%f %e\n", 100.0, 100.0);	//100.000000 1.000000e+02
System.out.print("%2.f\n", 123.5678);	//123.57
System.out.print("|%s|\n", "자바");	//|자바|
System.out.print("|%10s|\n", "자바");	//|        자바|
System.out.print("|%-10s|\n", "자바");	//|자바        |
System.out.print("%,d원\n", 1000000);	//1,000,000원
```
